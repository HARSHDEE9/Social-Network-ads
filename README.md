# Basic AI Chatbot

A simple desktop chatbot application built with Python. It uses predefined logic and conditional checks to respond to user input through an attractive graphical interface.

## Features

- Clean chat-style desktop user interface
- Rule-based responses for common questions
- Quick action buttons for easy testing
- Simple Python structure that is easy to understand and extend

## Project Structure

- `main.py` starts the application
- `ui.py` contains the Tkinter interface
- `chatbot.py` contains the chatbot response rules

## How to Run

1. Make sure Python 3.10 or later is installed.
2. Open the project folder in a terminal.
3. Run:

```bash
python main.py
```

## Example Queries

- `Hello`
- `What can you do?`
- `Tell me a joke`
- `What is the time?`
- `Describe the project`
